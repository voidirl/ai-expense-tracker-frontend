# Smart Expense Tracker — Frontend

React + Vite + Tailwind CSS frontend for the Smart Expense Tracker app.

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Chart.js + react-chartjs-2
- React Router DOM
- Framer Motion

## Setup
```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`. Requires backend at `http://localhost:8080`.
